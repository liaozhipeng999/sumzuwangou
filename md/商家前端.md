# 商家后台 · 接口文档

> **项目名称**：商家管理后台（Admin Merchant Houtai）  
> **前端技术栈**：Vue 3 + TypeScript + Vite + Element Plus + Pinia + Axios + WebSocket(STOMP/SockJS)  
> **后端 API**：http://localhost:9090（前端 dev proxy 路径前缀 /api）  
> **鉴权方式**：Authorization: Bearer {token}（请求头），token 存储于 localStorage['merchant_token']  
> **通用响应**：{ code: number, message: string, data: T }（code === 200 表示成功）

---

## 1. 商家鉴权模块 /merchant

### 1.1 商家登录
- POST /api/merchant/login
- 鉴权：否

请求体

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| loginId | string | 是 | 账号/用户名 |
| password | string | 是 | 密码 |

响应 data
```json
{
  "token": "eyJ...",
  "merchant": {
    "id": 1,
    "merchantName": "好又多超市",
    "merchantLogo": "https://.../logo.png",
    "merchantBrief": "社区便利",
    "username": "shop01",
    "contactName": "张三",
    "contactPhone": "13800138000",
    "email": "zs@example.com",
    "mainCategoryId": 2,
    "merchantLevel": 2,
    "status": 1,
    "sort": 1,
    "createdAt": "2025-03-01T10:00:00",
    "updatedAt": "2025-03-01T10:00:00"
  }
}
```

### 1.2 商家注册（入驻）
- POST /api/merchant/register
- 鉴权：否

请求体

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| merchantName | string | 是 | 店铺名 |
| username | string | 是 | 登录账号 |
| password | string | 是 | 密码 |
| contactName | string | 是 | 联系人 |
| contactPhone | string | 是 | 联系电话 |
| email | string | 否 | 邮箱 |
| merchantLogo | string | 否 | Logo URL |
| merchantBrief | string | 否 | 店铺简介 |
| mainCategoryId | number | 否 | 主营分类 ID |

响应 data：Merchant 对象（同上）

---

### 1.3 获取商家信息
- GET /api/merchant/info/{username}
- 鉴权：建议是
- 路径参数：username 登录账号

响应 data：Merchant 对象

---

### 1.4 退出登录
- POST /api/merchant/logout
- 鉴权：是

响应：{ code: 200, message: "ok", data: { message: "退出成功" } }

## 5. 客服会话模块 /customer-service

### 5.1 商家会话列表
- GET /api/customer-service/merchant/conversations
- 查询参数：shopId

响应 data：Conversation[]
```json
{
  "shopId": 1, "shopName": "好又多", "shopLogo": "...",
  "userId": 201, "userName": "李四",
  "lastMessage": "货到了吗？", "lastMessageType": "TEXT",
  "lastSendTime": "2026-06-29 10:05:00",
  "lastSenderType": "CUSTOMER", "unreadCount": 3
}
```

### 5.2 未读消息总数
- GET /api/customer-service/merchant/unread-count
- 查询参数：shopId
```json
{
  "total": 5,
  "users": [ { "userId": 201, "unreadCount": 3 } ],
  "shops": [ { "shopId": 1, "unreadCount": 0 } ]
}
```

### 5.3 商家标记某用户已读
- POST /api/customer-service/merchant/mark-read
- 请求体：{ shopId, userId }

### 5.4 商家发送消息
- POST /api/customer-service/merchant/messages

| 字段 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| shopId | number | 是 | |
| userId | number | 是 | |
| content | string | 是 | 文本/图片URL |
| messageType | string | 否 | TEXT/IMAGE/PRODUCT |
| productId | number | 否 | 商品卡片 |

响应 data：{ messageId, sendTime }

### 5.5 聊天记录分页
- GET /api/customer-service/messages
- 查询参数：shopId, userId, page=1, pageSize=20

响应 data：
```json
{
  "total": 80, "page": 1, "pageSize": 20,
  "list": [ {
    "messageId": 1001, "senderId": 201, "senderType": "CUSTOMER",
    "content": "你好", "sendTime": "2026-06-29 10:00:00", "messageType": "TEXT"
  } ]
}
```

### 5.6 快捷回复列表
- GET /api/customer-service/merchant/quick-replies
- 查询参数：shopId
- 响应：QuickReply[] → { id, content, sort }

### 5.7 新增快捷回复
- POST /api/customer-service/merchant/quick-replies
- 请求体：{ shopId, content, sort? }

### 5.8 删除快捷回复
- DELETE /api/customer-service/merchant/quick-replies/{id}
- 查询参数：shopId

### 5.9 上传消息图片
- POST /api/customer-service/upload/image
- Content-Type: multipart/form-data
- Form Data: file
- 响应：{ url, fileName }

### 5.10 标记单条消息已读
- PUT /api/customer-service/messages/{messageId}/read

## 6. WebSocket 实时通道（STOMP over SockJS）

- 连接：ws://localhost:9090/ws
- 库：@stomp/stompjs + sockjs-client
- 心跳 4000ms / 重连 5000ms

### 6.1 /topic/shop/{shopId}/messages
商家收到客户的新消息推送。

### 6.2 /topic/shop/{shopId}/unread
会话列表未读数角标实时更新。

### 6.3 /topic/user/{userId}/messages
用户维度新消息推送。

### 6.4 /topic/user/{userId}/unread
用户维度未读数通知。

消息体通用结构：
```json
{ "type": "NEW_MESSAGE", "shopId": 1, "userId": 201, "messageId": 1001, "content": "...", "sendTime": "..." }
```

## 7. 基础约定

### 7.1 前端代理（vite.config.ts）
```ts
server: { port: 5174, proxy: { '/api': { target: 'http://localhost:9090', changeOrigin: true, ws: true } } }
```

### 7.2 Axios 实例（src/api/merchant.ts）
- baseURL: /api、timeout: 10000
- 请求拦截器：注入 Authorization: Bearer {merchant_token}
- 响应拦截器：成功解包 response.data；401 清 token 并跳 /merchant/login

### 7.3 通用枚举
| 场景 | 值 | 说明 |
| --- | --- | --- |
| 订单 status | pending/paid/shipped/completed/cancelled | 待付款/已付款/已发货/已完成/已取消 |
| 商品 status | 0 / 1 | 下架 / 上架 |
| senderType | CUSTOMER / MERCHANT | |
| messageType | TEXT / IMAGE / PRODUCT | |
