# 电商后台管理系统 - 技术文档

## 一、项目概述

### 1.1 项目简介

本项目是一个基于 Vue 3 + TypeScript + Vite 构建的电商后台管理系统，主要面向商家提供店铺管理、商品管理、订单管理、客服中心等功能模块。系统采用现代化的前端技术栈，提供完整的商家入驻、商品运营、订单处理和客户服务能力。

### 1.2 技术栈

| 技术 | 版本 | 说明 |
|------|------|------|
| Vue | ^3.5.34 | 前端框架 |
| TypeScript | ~6.0.2 | 类型安全 |
| Vite | ^8.0.12 | 构建工具 |
| Vue Router | ^5.1.0 | 路由管理 |
| Pinia | ^3.0.4 | 状态管理 |
| Element Plus | ^2.14.1 | UI组件库 |
| Axios | ^1.17.0 | HTTP请求 |
| ECharts | ^6.1.0 | 数据可视化 |
| Stomp.js | ^7.3.0 | WebSocket客户端 |
| SockJS | ^1.6.1 | WebSocket兼容层 |

```typescript
router.beforeEach((to, _from) => {
  const publicPages = ['/login', '/merchant/login', '/merchant/register']
  const isPublicPage = publicPages.includes(to.path)
  const token = localStorage.getItem('merchant_token')
  const hasToken = !!token
  
  if (hasToken && isPublicPage) {
    return '/merchant/dashboard'
  } else if (!hasToken && !isPublicPage) {
    return '/merchant/login'
```

```typescript
async function login(username: string, password: string) {
  const cleanUsername = username.trim()
  const cleanPassword = password.trim()
  
  const res = await merchantLogin({ loginId: cleanUsername, password: cleanPassword })
  
  if (res.code === 200 && res.data) {
    token.value = res.data.token
    merchantInfo.value = res.data.merchant
    localStorage.setItem('merchant_token', res.data.token)
    localStorage.setItem('merchant_info', JSON.stringify(res.data.merchant))
    
    const now = new Date()
    const loginDate = `${now.getFullYear()}-${String(now.getMonth() + 1).padStart(2, '0')}-${String(now.getDate()).padStart(2, '0')}`
    localStorage.setItem('merchant_login_date', loginDate)
    
    return { success: true, message: res.message || '登录成功' }
  }
}
```

```typescript
stompClient = new Client({
```

```typescript
function getWebSocketUrl(): string {
  const protocol = window.location.protocol === 'https:' ? 'https' : 'http'
  const host = import.meta.env.DEV ? 'localhost:9090' : window.location.host
  return `${protocol}://${host}/ws`
}
```
