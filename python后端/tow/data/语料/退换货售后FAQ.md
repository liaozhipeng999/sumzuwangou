# 退换货与售后FAQ

## 退换货政策

Q: 商品可以退换吗？
A: 符合以下条件的商品可申请退换：
1. 商品在7天无理由退换期内（特定商品除外）
2. 商品保持原包装完好，未使用
3. 商品配件、赠品齐全
4. 非定制类商品

注意：以下商品不支持7天无理由退换：
- 食品、保健品
- 个人卫生用品
- 定制商品
- 虚拟商品（充值卡、软件等）

Q: 如何申请退货？
A: 退货申请流程：
1. 登录账户进入"我的订单"
2. 选择需要退货的订单
3. 点击"申请售后"，选择退货原因
4. 提交申请等待审核
5. 审核通过后按指引寄回商品
6. 商家收到商品后退款

Q: 退货需要自己承担运费吗？
A: 运费承担规则：
- 商品质量问题：商家承担运费
- 个人原因退货：买家承担运费（可使用运费险抵扣）
- 7天无理由退货：买家承担运费

## 退款流程

Q: 退款多久能到账？
A: 退款到账时间：
- 微信/支付宝支付：1-3个工作日
- 银行卡支付：3-7个工作日
- 优惠券/积分：退款成功后立即返还

Q: 退款金额不对怎么办？
A: 退款金额为实际支付金额（扣除已使用的优惠券、运费险等）。如有疑问请联系客服核实。

## 售后服务

Q: 商品出现质量问题怎么办？
A: 质量问题处理流程：
1. 拍照或视频记录问题
2. 联系客服说明情况
3. 根据问题程度选择维修、换货或退货
4. 按客服指引完成售后流程
